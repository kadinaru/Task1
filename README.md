# Task1
----------------
**Описание**:  Это приложение позволяет выбрать цвет и контур, координаты и размеры, а также рисовать ёлку. Также можно нарисовать фигуру с эффектами.
 - **Технологический стек**: JavaFX.
 - **Статус**:  1.0.
